## git init
	1、把当前目录初始化为版本库
	2、当前目录下生成一个隐藏文件.git


## git add 文件名
	1、把当前目录下的某个文件提交到暂存区
    2、 git add readme.md 把这个文件提交到暂存区
    3、 git add . 把所有文件提交到暂存区
